# Painel Gerencial Mockup Project

Projeto React + Vite com um mockup visual da aba **Painel Gerencial**, focado em:

- gráficos mais premium
- menos espaço vazio
- melhor hierarquia executiva
- rankings mais fortes
- fallback elegante para pouca base

## Como rodar

```bash
npm install
npm run dev
```

## Objetivo

Este projeto é uma **referência visual** para o Codex aplicar no sistema real.

### O que ele mostra
- hero refinado da aba Painel Gerencial
- KPIs executivos com delta
- tendência diária em área/linha
- comparativo com período anterior em barras verticais compactas
- `Mapeado x não mapeado` em donut + barras compactas
- rankings horizontais mais sofisticados
- estado alternativo de pouca base

### Observação
Não é integração funcional com backend. É apenas direção de UI e composição visual.
